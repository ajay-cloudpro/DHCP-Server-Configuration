# DHCP Server Configuration

## Overview

This project demonstrates the configuration of a DHCP Server in Cisco Packet Tracer. The DHCP server is configured to automatically assign IP addresses and network settings to client devices.

## Project Objectives

- Configure a DHCP Server
- Create a DHCP Pool
- Assign IP addresses automatically
- Configure subnet mask
- Configure default gateway
- Configure DNS server
- Verify successful IP assignment

## Configuration Details

| Setting | Value |
|----------|---------|
| Network Address | 192.168.1.0 |
| Subnet Mask | 255.255.255.0 |
| Default Gateway | 192.168.1.1 |
| DNS Server | 8.8.8.8 |

## Procedure

1. Add a Server, Switch, and PCs in Cisco Packet Tracer.
2. Connect all devices using appropriate cables.
3. Configure the DHCP service on the server.
4. Create a DHCP pool and define:
   - Network address
   - Subnet mask
   - Default gateway
   - DNS server
5. Configure PCs to obtain IP addresses automatically.
6. Verify that each PC receives a valid IP address from the DHCP server.

## Verification

- DHCP server successfully assigns IP addresses.
- Clients receive gateway information.
- Clients receive DNS configuration.
- Network connectivity is verified.

## Tools Used

- Cisco Packet Tracer
- DHCP Protocol
- TCP/IP Networking
